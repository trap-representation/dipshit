A word from the authors

Why we decided to make Dipshit, you might ask. Or may not. But we'll answer anyway because this is our website... of course, in case you weren't already aware of that. So, simplicity isn't always proportional to how simple it is to program a computer; it is usually quite the opposite. It is how simple the actual computer is, and this is excatly what YSM tries to achivee: simplicity. Dipshit is an implementation of the YSM ISA that makes writing portable applications for the web simpler without any of the bloat that comes with modern tools.
